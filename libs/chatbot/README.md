# Chatbot Library (chatbot)

The `chatbot` library is an auxiliary utility package designed to encapsulate and abstract the LangChain framework.

## What it is

This package provides reusable classes, unified factory initializers for different LLM providers (OpenAI, Ollama, Qwen, etc.), and standardized prompting templates. It serves as the code foundation used to build the various chatbot agents across this repository.

## For what it can be used for

- Standardizing LLM model initialization across applications and scripts.
- Declaring custom system prompts and tools binding parameters in a unified way.
- Accelerating the development of new chatbot layouts or testing setups without duplicating boilerplate code.
