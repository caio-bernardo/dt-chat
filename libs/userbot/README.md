# Userbot Library (userbot)

The `userbot` library is a robust simulation library designed to act as a human client.

## What it is

This package creates autonomous customer simulation agents (UserBots) using LangChain. Endowed with unique names, ages, behavioral personas (e.g. anxious, aggressive, patient), and goals (e.g. contesting charges, applying for credit cards), these bots interact with conversational assistants and emulate human chat timings (typing, thinking, breaks) using the `timesim` library.

## For what it can be used for

- Simulating realistic multi-turn human dialogues with Bancobot or other assistant models.
- Stress-testing chatbots with challenging client personalities and goals.
- Backing high-volume parallel client swarms used during system evaluations and process discovery.
