# dtchat
Framework for Digital Twins of conversational agents.

## Features

### Functional Requirements
- [ ] Permitir a criação de chatbots customizáveis (pode mudar a engenharia de prompt, mudar os hiperparâmetros, o modelo)
- [ ] Permitir a criação de chatbots com engenharia de prompt customizavel, modelo e outras configurações.
- [ ] Json-API de Chatbot bancário
- [ ] Chatbot de clientes
- [ ] Criação de um log de eventos com processamento de touchpoints

## Pre-requisites
- Python +3.12
