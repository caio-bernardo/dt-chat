# Visualizer

Visualizer for the BancoBot interactions via a web-based interface, allowing to view and export conversations.

## Features
- UI to view all conversations, split by session
- [ ] See AI messages and human messages
- [ ] Export messages in different formats: csv, json
- [ ] Realtime updates
